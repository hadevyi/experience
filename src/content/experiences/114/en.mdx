    I developed SSAFYnity's official website and blog. The website handles introduction, membership, and inquiry flows, while the blog keeps notices and operational records.

    As the 5th president, I needed to organize information that had been scattered across KakaoTalk, SNS channels, and event announcements. Records such as events and dues also needed a searchable blog.

    I separated the two sites because their metrics and Google Analytics / Search Console criteria could differ by purpose.
  scope: |

    - Blog: Markdown content archive for notices, journals, clubs, series, and dues records
    - Operational data: member statistics by cohort and campus, event data, operator data, club data, and SNS links
    - Analytics and search: separated Google Analytics and Google Search Console criteria by site purpose
    - Deployment: static build and GitHub Pages deployment through GitHub Actions
  contribution: |

  implementation: |
    The official website uses Astro file-based routing. Each route generates static HTML, and operational data is separated into TypeScript files.
