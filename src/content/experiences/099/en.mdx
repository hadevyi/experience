    I participated in the 9th cohort of Open Source Contributors for one month in August 2025.
    The activity was conducted online, mainly through Discord and KakaoTalk.
  learning: |
    There were no fixed weekly missions, but guidance was provided so participants could work toward the goal within the period.
    Each participant chose open source projects based on their own interests and goals.
    I used practical relevance in the field and alignment with my career roadmap as the main criteria.

    For the repositories I selected, I first reviewed their guides, first issues, and issues suggested as contribution guides.
    I then used AI to analyze the code structure, learning how to understand and work with open source more effectively.
  challenge: |
    It took time to learn how to read the repository structure and issue context of the open source projects I selected.
    The challenge was not only reading the code itself, but also understanding how the guide documents and issues connected to the contribution flow.
