  overview: |
    HOMEZ was a web project that helped users search Seoul housing transaction data and choose the information they needed when looking for a home. It also included administrator statistics that could support future advertising or operation decisions.
  background: |
    This was the final project of the first semester in SSAFY 5th cohort. The housing transaction theme was given as part of the assignment, and the project required connecting frontend, backend, database, and external APIs within a limited timeline.
  scope: |
    The project implemented housing transaction search, Kakao Map based location information, notices, member features, admin features, and statistics visualization.
  contribution: |

Based on the repository commit history, I mainly contributed to member-related backend APIs and Vue screens, common code, router/store integration, and some UI refinements. I worked on member features and admin features, including a member list screen and charts for new registrations, age distribution, and gender ratio. The Spring Boot and Vue project structure, as well as database design and implementation, were handled together by the team.
